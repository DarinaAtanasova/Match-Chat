# Match&Chat
Web application for matching users with similar interests.

Used technologies:
+ Node.js
+ Express.js
+ Socket.io
+ Bootstrap v3.4
+ MongoDB

## How to use

**_Install dependencies:_**

```shell
npm install
```

**_Run project:_**

```shell
npm start
```

**_Run in dev mode:_**

```shell
npm run dev
```
